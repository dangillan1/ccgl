# CCGL GrowLink Analyst

Automated cannabis cultivation monitoring for Cape Cod Grow Lab.

## Usage

```bash
python3 generate_report.py
```

## Structure

- generate_report.py — Report generator
- report_template.html — HTML template
- data/state.json — Sensor readings
- data/logo_base64.txt — Logo
- data/wordmark_base64.txt — Wordmark
